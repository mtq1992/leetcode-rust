* lib.rs 文件 声明每个算法文件。

## Benchmark

### rust 自带 bench
使用官方 Benchmark 需要安装 nightly 版本rust。

安装命令  
`rustup install nightly`

在项目根目录下切换成nightly  
`rustup override set nightly`

`cargo bench --lib two_sum`


### Criterion

需要在 Cargo.toml 添加如下内容  
```
[[bench]]
name = "two_sum"
harness = false

[[bench]]
name = "three_sum"
harness = false

```  
多个 `bench` 声明多个目标。  
`harness = false` 这里告诉编译器关闭 libtest benchmark
